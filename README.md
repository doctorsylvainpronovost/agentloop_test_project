# Weather Web App Scaffold

Minimal full-stack scaffold in repository root:

- Frontend: TypeScript + Vite
- Backend: FastAPI + Uvicorn + httpx

## Install

```bash
npm install
python3 -m pip install -r requirements.txt
```

## Run

Frontend dev server:

```bash
npm run dev
```

Backend dev server:

```bash
npm run backend:dev
```

## Build

```bash
npm run build
```

## Tests

```bash
npm test
```
